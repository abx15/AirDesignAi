# Security Policy

## Supported Versions

We currently support the following versions of MotionMath AI:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of MotionMath AI seriously. If you believe you have found a security vulnerability, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them via email to `security@motionmath.ai`.

Please include:

- A description of the vulnerability.
- Steps to reproduce the vulnerability.
- The potential impact if exploited.

We will acknowledge receipt of your report within 48 hours and provide a timeline for addressing the issue.
